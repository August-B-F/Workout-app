# Astra

Astra is a workout tracker I made in Flutter. You build a workout as a list of timed tasks, run it through a timer, and it keeps track of how often you train. Everything is stored locally with Hive, so there is no account and no server. There is a screen for building workouts, a timer screen that runs one with a progress bar and a skip button, and a statistics screen that shows your training over the week and month as small sparklines. Run it with flutter pub get and then flutter run.
